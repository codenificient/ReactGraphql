# React GraphQL

A full-stack application with React frontend and GraphQL API backend, demonstrating modern client-server architecture.

## Project Structure

- `client/` — React frontend
- `frontend/` — Alternative frontend implementation
- `server/` — GraphQL API server

## Getting Started

```bash
# Start the server
cd server && npm install && npm start

# Start the client
cd client && npm install && npm start
```
